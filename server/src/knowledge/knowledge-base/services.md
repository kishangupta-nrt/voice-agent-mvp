# Services

## Web Development
- **Landing Pages**: Single-page marketing sites with animations, contact forms, and SEO. Built with Next.js or React. Typical timeline: 1-2 weeks.
- **Business Websites**: Multi-page sites with CMS, blog, and admin dashboard. Built with Next.js + Sanity/Strapi. Typical timeline: 3-4 weeks.
- **Web Applications**: Full-stack apps with user auth, payments, real-time features. Built with Next.js, Supabase, Stripe. Typical timeline: 4-8 weeks.
- **E-commerce**: Online stores with product catalog, cart, payment, order tracking. Built with Next.js + Shopify/Medusa. Typical timeline: 4-6 weeks.

## App Development
- **Mobile Apps (iOS + Android)**: Cross-platform apps built with React Native or Flutter. Includes push notifications, offline mode, and app store submission. Typical timeline: 6-12 weeks.
- **MVP Apps**: Minimal viable product for startups - core features only, fast launch. Built with React Native + Supabase. Typical timeline: 3-4 weeks.

## AI Solutions
- **Chatbots & Voice Agents**: Custom AI assistants for customer support, sales, or internal use. Built with LLM APIs, RAG, and voice synthesis. Typical timeline: 2-4 weeks.
- **AI Automation**: Workflow automation using AI - document processing, email sorting, data extraction. Typical timeline: 2-6 weeks.
- **Custom AI Models**: Fine-tuned models for specific business use cases (classification, generation, recommendation). Typical timeline: 4-8 weeks.

## Add-ons
- **SEO Optimization**: On-page SEO, meta tags, sitemap, performance optimization.
- **Analytics Dashboard**: Custom dashboards with real-time metrics and reports.
- **API Development**: REST or GraphQL APIs for third-party integrations.
- **Maintenance & Support**: Ongoing bug fixes, updates, and feature additions (monthly retainer).
