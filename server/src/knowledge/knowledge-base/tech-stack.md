# Tech Stack

## Frontend
- **React** - Component-based UI library for web apps
- **Next.js** - Full-stack React framework with SSR, SSG, and API routes
- **React Native** - Cross-platform mobile app development
- **Tailwind CSS** - Utility-first CSS framework for rapid styling
- **TypeScript** - Type-safe JavaScript for better code quality

## Backend
- **Node.js** - Runtime for server-side JavaScript
- **Express / Fastify** - API frameworks for Node.js
- **Supabase** - Backend-as-a-Service with Postgres, Auth, Storage, Realtime
- **PostgreSQL** - Relational database with advanced querying
- **Redis** - Caching and real-time data processing

## AI / ML
- **OpenAI GPT-4 / GPT-4o** - Language model for chat, generation, and reasoning
- **Mistral** - Open-weight language model for cost-effective deployments
- **LangChain** - Framework for building LLM-powered applications
- **RAG (Retrieval-Augmented Generation)** - Custom knowledge base integration
- **Whisper** - Speech-to-text transcription
- **ElevenLabs / Coqui TTS** - Text-to-speech synthesis

## DevOps & Infrastructure
- **Vercel** - Hosting for Next.js and frontend deployments
- **AWS / GCP** - Cloud infrastructure for backend services
- **Docker** - Containerization for consistent environments
- **GitHub Actions** - CI/CD pipeline automation

## Third-party Integrations
- **Stripe** - Payment processing and subscriptions
- **Razorpay** - Payment gateway for Indian market
- **SendGrid / Resend** - Email delivery
- **Twilio** - SMS, voice, and WhatsApp messaging
- **Shopify / Medusa** - E-commerce platforms
